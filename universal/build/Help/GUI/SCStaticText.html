SCStaticText


A non-editable textfield


string_(string)

set the text.


font_(font)

set the font.


stringColor_(color)

set the color of the string.


Examples

w = SCWindow.new.front;

a = SCStaticText(w, Rect(10, 10, 100, 20));

a.string = "Rolof's Rolex";

)


// adjust bounds

a.bounds = Rect(5, 5, 100, 20)
