TabbedLayout


multiple layouts switched by clicking a tab button to cycle through them


TabbedLayout( parent, bounds, tabs )


parent: nil, a view or a layout

bounds: nil, a rect, or a point (width by height)

tabs : titles and layout functions

[ title, function, title2, function2 ... ]


(


TabbedLayout(nil,200@250,


[

"one" , { arg layout;

ActionButton(layout,"one----")

},

"two" , { arg layout;

SCSlider(layout,20@200);

ActionButton(layout,"two----")

}

]

)


)


.select(i)

programmatically select to view one of the tabs

selected

the index of currently selected


<>action

action evaluated when the tab is changed by the gui (not by select() )
