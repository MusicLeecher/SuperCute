SuperCollider Help


Help files are HTML files kept in the Help directory in the SuperCollider application folder. 

If you're an absolute beginner please also refer to the README that came with the application download.


A help browser can be opened by executing the following line of code. To execute code, put the cursor anywhere on the line and press:  enter, not return (OS X) alt+enter (Windows) Ctrl+c Ctrl+c (Emacs) Ctrl+l (scfront) (for key commands, also see Shortcuts)


Help.gui; //to run a single line of code put the cursor anywhere on the line and execute


Shortcut links within this file: click them to explore, or ask for help after selecting them. To obtain help on an item, use your Help menu, or a particular shortcut for your platform:  cmd+d (OS X) F1 (windows) Ctrl-c Ctrl-h (Emacs)


SuperCollider uses classes, whose names all begin with a capital letter. The help file for a class can be called up by selecting the text of a class name, then asking for help. You can also browse the class directory: 


Object.browse  //run this line of code
